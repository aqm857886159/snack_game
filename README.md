# 优雅贪吃蛇 (Elegant Snake)

一个具有现代设计感的贪吃蛇网页游戏。

## 功能特点

- 🎮 简单直观的游戏控制
- 🎨 优雅的苹果风格界面设计
- 📊 实时分数显示
- 🏆 本地排行榜系统
- 💾 自动保存游戏记录
- 📱 响应式设计，支持各种设备

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage (用于数据持久化)

## 如何运行

直接在浏览器中打开 `index.html` 文件即可开始游戏。

## 游戏控制

- 使用方向键或 WASD 控制蛇的移动
- 点击"开始游戏"按钮开始
- 点击"重新开始"按钮重置游戏
- 游戏结束后自动保存分数
